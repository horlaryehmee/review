== Installation ==

Here are few steps to follow:

1. Upload `gs-facebook-comments` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Select Facebook Comments page from wordpress menu for settings.
4. Select post types or use shortcode [gs-fb-comments] for pages or manuall comments.
5. That's it. You're ready to go!

== Frequently Asked Questions ==

= Can I show Facebook comments on pages? =

Yes you can use shortcode [gs-fb-comments] for pages or anywhere at your site.

= Can I use WP Social Comments without Facebook App ID? =

Yes you can use it without Facebook App ID. But you won't be able to do any comments moderation tools.
